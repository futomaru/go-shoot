Go Shoot

このプロトタイプは、P2P ブラウザシューティングゲーム「Go Shoot」の バックエンド API とロビー WebSocket を Go のみで構築するための土台です。

API は cmd/server から起動でき、ゲームロビーの作成や参加、簡易的な チャットブリッジを提供しています。

クイックスタート

  1. go run ./cmd/server を実行
  2. 別ターミナルで curl -X POST localhost:8080/api/v1/rooms
  3. ws://localhost:8080/ws/lobby/<roomID> に接続してチャット メッセージを送信
